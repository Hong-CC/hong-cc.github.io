Connect Four

  • 題目點我

  • Task：Connect Four是一個經典的桌遊（就是Gif那個遊戲）。

    題目的Input是一個List，並限定在row = 6、column = 7的棋盤大小，任務目標是找出哪個顏色獲勝。其中Input的List大概長這樣：[“A_Red”, “B_Yellow”, “A_Red”, “B_Yellow”, “A_Red”, “B_Yellow”, “G_Red”, “B_Yellow”]，首字母是落子在哪一個column，顏色就是落子的顏色。另外，題目作者有說，List裡的紀錄，不會因有選手獲勝而停止（所以不能抓最後一筆是什麼顏色作為判斷誰獲勝的條件）

  • 解題邏輯：

    1. 先用numpy建立一個6x7的矩陣當作棋盤。

    2. 落子部分先用ASCII將A變0、B變1，這樣旗子要到哪個Column就解決了。Row的部分則利用for搭配if從最後一Row開始檢查，如果檢查的那個位子是0就落子，如果不是0就往上一Row。
      這邊要注意：for i in range(5, -1, -1)終結要是-1，這樣Row 0才會算進去。

    3. 判斷勝負的地方，就拆開成四種狀況討論，垂直方向、水平方向、斜向、反斜向，這部分就是for搭配矩陣為字的應用，矩陣位置數對就不太難。

  • 小心得：

    1. 缺點是用numpy建矩陣，沒辦法放入str，所以必須先把顏色先轉成1 or 2，最後return的時候還要轉回顏色，有點多此一舉。
    2. 判斷贏手的方法是拆開各種狀況討論，感覺有點瑣碎，應該有更聰明的解法，在codewars的解答裡有看到用卷積2D（Convolution 2D)的方法解題，超猛。

ParseInt Reload

  • 題目點我

  • Task：給定字串轉換成數字，其中，最小的數字是0，最大的是1 million（我比較爛覺得這大幅地降低難度）。

    舉例： “one” => 1；

    ​ “two hundred forty-six” => 246；

    ​ “seven hundred eighty-three thousand nine hundred and nineteen” => 783919

  • 解題邏輯：

    1. 一些比較特殊的字，先建成dictionary。
    2. 字串分割成List，用for一個一個讀進去，例如eighty-three thousand nine hundred and nineteen，會先讀到eighty並被記錄在tmp，接著讀到three，更新tmp變成$80+3$，接著讀到thousand，表示tmp裡的數字是千位數（後面要有3個0），所以tmp$\times 1000$，此時千位數以上的數字都完成了，所以讓tmp歸零，接著按照前面的邏輯完成百位數。
    3. 需要注意的是10萬這個量級，英文是one hundred thousand，這邊的hundred就不是tmp$\times 100$，是tmp$\times 100,000$

  • 小心得：

    1. 如果字串分割切點有多個可能，用re.split()正規化的方式比較方便，像是two hundred forty-six要拆成[“two”, “hundred”, “forty”, “six”]，就必須同時使用空格（” “）以及短線（”-“）分割字串。
    2. 自己解題的邏輯還是比較直線思考，所以遇到10萬這個數字時就踢到鐵板了，最後只能算接下來的字串有沒有出現thousand，因為先出現hundred，再出現thousand，代表這個數字一定>10萬，雖然問題是解決了，但要擴展感覺比較困難，像是two hundred million, forty-five thousand six hundred seventy-eight(200,045,678)，就會出問題了。

  • Task：給定一串整數列和一個整數（和），求出數列中加起來等於「和」的兩個數。

    舉例：$([3, 11, 7, 5], 10)$ → 應該得到$[3, 7]$，因為$3+7=10$

Mixed Number Converter

  • 題目點我

  • Task：給一個型態是字串的分數，得到一個帶分數，同時分數部分，必須是最簡分數。

    舉例：

    $”42/9”$ → $”4\ \ 2/3”$

    $”6/2”$ → $”3”$

    $”-10/7$ → $”-1 \ \ 3/7”$

    $”x/y$ → $”a\ \ b/c”$

  • 解題邏輯：先處理分母 or 分子是0，還有整除的狀況先寫好。

    其他狀況如下：

    $
    \frac{x}{y} = a + \frac{b}{c}
    $
    $\frac{b}{c} = \frac{x}{y}-a = \frac{x - a \times y}{y}$
    其中：
    $a = x / y的整數部分$
    $b = x - a \times y$
    $c = y$

    最後針對b、c利用最大公因數對其進行約分。
